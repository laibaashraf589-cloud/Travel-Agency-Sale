# Travel Agency Sales Dashboard

## 📊 Project Overview
This project contains an Excel dashboard created for Travel Agency data.  
The dashboard helps analyze sales by:
- Destination
- Package Type
- Payment Status
- Booking & Travel Dates

## 🛠️ Skills Used
- Pivot Tables
- Pivot Charts
- Slicers & Timelines
- Dashboard Formatting
